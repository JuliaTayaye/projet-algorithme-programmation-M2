Python est un langage de programmation en open source interprété dont la première version est apparue le 20 Février 1991 par l’auteur Guido Van Rossum. Il est disponible sous une licence libre : Python Software Foundation License. Aujourd’hui, nous en somme à la version 3.0 et il se pourrait que la version 4.0 ne voit jamais le jour selon les dire de son fondateur, à cause des difficultés rencontrées lors de la migration de python 2.0 à python 3.0 en 2008. Il est le langage de programmation open source le plus employé par les informaticiens. Lorsqu’on parle de la gestion d’infrastructure, d’analyse de données ou du développement de logiciels, on y associe Python. L’objectif de sa création était d’automatiser les éléments les plus ennuyeux de l’écriture de scripts ou de réaliser rapidement des prototypes d’applications. Pour les développeurs, Python est utilisé principalement pour la :

  • Programmation d’applications
  • Création de services web
  • Génération de code
  • Métaprogrammation
